===============================================================================

　MemoryLamp

===============================================================================


■ソフトの概要

　タスクトレイに物理メモリの使用率を表示するプログラムです。
　使用率に応じて１０種類のアイコンを表示します。
　アイコンは .ico ファイルから読み込んでいるので、ファイルを差し替えて好きなアイコンに変更することができます。


■動作環境

　Windows 10 Pro または Windows 10 Home


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■起動方法

　MemoryLamp.exe を実行して下さい。


■終了方法

　タスクトレイ上のアイコンを右クリックして「終了」を選択して下さい。


■アイコンファイル一覧

　MemoryUsage_0-10_Pct.ico     ... 物理メモリ使用率 10 % 未満
　MemoryUsage_10-20_Pct.ico    ... 物理メモリ使用率 10 % 以上 20 % 未満
　MemoryUsage_20-30_Pct.ico    ... 物理メモリ使用率 20 % 以上 30 % 未満
　MemoryUsage_30-40_Pct.ico    ... 物理メモリ使用率 30 % 以上 40 % 未満
　MemoryUsage_40-50_Pct.ico    ... 物理メモリ使用率 40 % 以上 50 % 未満
　MemoryUsage_50-60_Pct.ico    ... 物理メモリ使用率 50 % 以上 60 % 未満
　MemoryUsage_60-70_Pct.ico    ... 物理メモリ使用率 60 % 以上 70 % 未満
　MemoryUsage_70-80_Pct.ico    ... 物理メモリ使用率 70 % 以上 80 % 未満
　MemoryUsage_80-90_Pct.ico    ... 物理メモリ使用率 80 % 以上 90 % 未満
　MemoryUsage_90-100_Pct.ico   ... 物理メモリ使用率 90 % 以上

　※アイコンファイルは MemoryLamp.exe と同じ場所に置いて下さい。


■取り扱い種別
