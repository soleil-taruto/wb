===============================================================================

　AccessLamp_D

===============================================================================


■ソフトの概要

　ディスク（外部記憶装置）毎のアクセス状況を表示します。


■動作環境

　Windows 10 Pro または Windows 10 Home


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■起動方法

　AccessLamp_D.exe を実行して下さい。


■終了方法

　当アプリのウィンドウを右クリックして「終了」を選択して下さい。


■表示の見かた

　上段 ... 読み込み
　下段 ... 書き出し


■監視対象インスタンスを指定する方法

　当アプリのウィンドウを右クリックして「設定」を選択して下さい。
　監視対象にしたいインスタンス名を選択状態にして、設定ダイアログを閉じて下さい。


■取り扱い種別

　フリーソフト
