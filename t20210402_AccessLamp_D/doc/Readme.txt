===============================================================================

　AccessLamp_D

===============================================================================


■ソフトの概要

　ディスク（外部記憶装置）毎のアクセス状況を表示します。


■動作環境

　Windows 10 Pro または Windows 10 Home


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■起動方法

　AccessLamp_D.exe を実行して下さい。


■終了方法

　当アプリのウィンドウを右クリックして「終了」を選択して下さい。


■画像ファイル一覧

　Denied.png        ... アクセス拒否・無効
　Idle.png          ... アクセス無し
　Busy.png          ... 読み書き中
　VeryBusy.png      ... 読み書き中（ビジー状態）

　※200ミリ秒以上持続的又は断続的にアクセスが続いた場合、ビジー状態と判断します。

　※画像ファイルは実行時のカレントディレクトリ又は AccessLamp_D.exe と同じ場所に置いて下さい。


■監視対象インスタンスを指定する方法

　当アプリのウィンドウを右クリックして「設定」を選択して下さい。
　監視対象にしたいインスタンス名を選択状態にして、設定ダイアログを閉じて下さい。
