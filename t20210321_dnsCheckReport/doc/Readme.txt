マップファイル変換
==================


----
概要

Elsa-系マップファイルを Platinum v1.81 で編集可能な形式に変換とその逆を行います。


----
起動方法

MapConverter.exe を実行して下さい。


----
変換 / Elsa-系マップファイル --> Platinum v1.81

マップファイル (拡張子 .txt) を、ウィンドウ上部の領域にドラッグアンドドロップして下さい。

成功すれば、マップファイルと同じ場所に Platinum 向けマップデータフォルダが作成されます。


----
変換 / Platinum v1.81 --> Elsa-系マップファイル

マップデータフォルダまたはマップデータ (拡張子 .csv) を、ウィンドウ上部の領域にドラッグアンドドロップして下さい。

成功すれば、マップデータフォルダと同じ場所にマップファイル (拡張子 .txt) が作成されます。
